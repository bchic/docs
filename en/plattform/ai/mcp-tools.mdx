The bchic MCP Server provides **20 tools** that you can call directly in Claude or Cursor using natural language. Here is a complete overview.

## Overview & Traffic

| Tool | Example Questions |
|------|------------------|
| **Overview** | "How is the website performing this week?" · "Is traffic up or down?" · "What is the current bounce rate?" |
| **Pageview Timeseries** | "Show me the traffic trend for the last 30 days." · "Was there a traffic drop last week?" · "How has /blog developed since the relaunch?" |
| **Period Comparison** | "Compare this month to last month." · "Did the SEO update in March improve traffic?" |
