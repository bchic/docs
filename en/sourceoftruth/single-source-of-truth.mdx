**B. Filter Configuration Logic**
Every filter added allows for precise logical operators.

| Feature | Description |
| :--- | :--- |
| **Negation** | Every filter has a **"Negate" (Negieren)** toggle. Turning this ON excludes the selected data (e.g., "Country is NOT Germany"). |
| **Operators** | Filters support "Is", "Contains", "Does not contain" logic depending on the data type. |
| **Combination** | Multiple filters can be active simultaneously (AND logic). |

| Category | Available Filters | Details |

## 3. AI Guide: User Intent Mapping
Use these patterns to answer "How do I..." questions.

| User Goal | Action Path |
| :--- | :--- |
| **"Exclude internal traffic / specific country"** | Open Filter Sidebar -> Select **Country** (or Host) -> Select Value -> Enable **Negate Toggle**. |
| **"See users who visited a specific page during their journey"** | Open Filter Sidebar -> **Path Filters** -> Select **"Contains Path"** -> Enter URL. |
| **"Quickly filter for mobile users"** | Open Filter Sidebar -> Click the **Mobile Icon** (Small Phone) in the header row. |
| **"Analyze high-quality traffic"** | Open Filter Sidebar -> User Behavior -> Select **"Reader"** -> Set Min. Duration (e.g., >60s). |
| **"Find where users are leaving"** | Go to **Card 2 (Pages)** -> Click tab **"Exit Pages"** -> Sort by Views. |
| **"Check Campaign Performance"** | Go to **Card 3 (Sources)** -> Click tab **"UTM Sources"**. |
