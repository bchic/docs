### Filterleiste
Wie auf anderen Seiten können Sie auch hier die Daten nach `Zeitraum` und mit `Spezialfiltern` eingrenzen, um beispielsweise nur die Klicks von einer bestimmten Unterseite Ihrer Website aus zu analysieren.

### Liste der ausgehenden Link-Klicks
Das zentrale Element dieser Seite ist eine einfache, aber aussagekräftige Liste. Sie enthält zwei Spalten:

*   **URL:** Die exakte externe URL, auf die der Nutzer geklickt hat.
*   **Anzahl:** Die Gesamtzahl der Klicks auf diese spezifische URL im ausgewählten Zeitraum.

Die Liste ist standardmäßig absteigend nach der Anzahl der Klicks sortiert, sodass die populärsten externen Links immer ganz oben stehen.
